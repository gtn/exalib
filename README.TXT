exalib
===============
GTN-Solutions

This plugin is a simple library where uploads of documents and other resources can be done course-independently by admin available to everyone.
This block is for Moodle 2.7+ versions, it will not work for versions below 2014051200


* exalib - Bugs, Feature Requests, and Improvements *

If you have any problems installing this block or suggestions for improvement please mailto: office@gtn-solutions.com

* exalib - Description *
The block can be accessed from within any course. The admin can add any kind of document using the "manage library" link of the block.
New entries can be made with references to the source, links, file, content and author. Also categories can be defined with the link "manage categories".
With the search box the library can be searched as a whole or within each category.

* exalib - Disclaimer *

As with any customization, it is recommended that you have a good backup of your Moodle site before attempting to install contributed code.
While those contributing code make every effort to provide the best code that they can, using contributed code nevertheless entails a certain degree of risk as contributed code is not as carefully reviewed and/or tested as the Moodle core code.
Hence, use this block at your own risk.

* exalib - Installation *

1) Save the zip file somewhere onto your local computer and extract all the files

2) Transfer the folder exalib to the blocks-directory of Moodle

3) Optionally install the repository plugin: Copy moodle_path/blocks/exalib/repository_plugin/exalib folder to moodle_path/repository/exalib

4) Log in as 'administrator' and click on the 'Home' link

That's it!
